goAML Report Types — UAE FIU

Official report types per UAE FIU goAML Portal (v1.2, April 2024) — uaefiu.gov.ae | EOCN TFS: uaeiec.gov.ae | TFS Inquiries: tfs@eocn.gov.ae

STR
Suspicious Transaction Report
Suspicious executed transaction linked to ML/TF. File via goAML immediately, without any prior consent.
SAR
Suspicious Activity Report
Suspicious activity or attempted / non-executed transaction. File when no transaction occurred but behaviour is suspicious.
FFR / CNMR
Funds Freeze Report / Confirmed Name Match Report
Filed when funds/assets are frozen under TFS. CNMR is the updated EOCN term for FFR. Submit via goAML within 5 business days of implementing freeze. (Cabinet Decision 74/2020, Art.21)
PNMR
Partial Name Match Report
Partial hit on sanctions/watchlist name. Entity has 10 business days to verify/discount. If unresolved: reject transaction, then file PNMR within 5 business days of rejection. (Cabinet Decision 74/2020)
DPMSR
Dealers in Precious Metals & Stones Report
DPMS-specific reporting for suspicious or threshold-based activities. AED 55,000+ cash transactions for precious metals/stones.
HRC
High Risk Country Transaction Report
Transaction involving a FATF-listed or FIU-designated high-risk country (Black/Grey list jurisdictions).
HRCA
High Risk Country Activity Report
Activity or attempted activity involving a high-risk country. No completed transaction required.
AIF
Additional Information File (no transactions)
Response to FIU request for additional information (clarifications, documents, KYC) without transaction data.
AIFT
Additional Information File (with transactions)
Response to FIU request for additional information that includes transaction data.
RFI
Request for Information (no transactions)
Response to FIU mass information request to multiple reporting entities without transaction data.
RFIT
Request for Information (with transactions)
Response to FIU mass information request that includes transaction details.
REAR
Real Estate Activity Report
Real estate-related activities including virtual asset-linked property transactions.
